## 目录使用规则

1. 新一轮审查新增独立报告，不覆盖历史报告正文。
2. 审查报告负责“现象、证据、影响、状态、动作”，不重复写实现细节。
3. 可执行跟踪项必须同步到 `docs/issues/`，避免审查结论孤立。
