| 状态 | 图标 | 说明 |
|-----|------|------|
| ✅ 已完成 | 🟢 | 设计文档完整，已评审通过，可以进入实施阶段 |
| 🟡 进行中 | 🟡 | 设计文档正在编写或完善中 |
| 🔴 未开始 | 🔴 | 尚未开始设计或仅有草稿 |
| ⏸️ 已搁置 | ⚪ | 暂时搁置，不在当前开发计划中 |
| 🔄 需更新 | 🔵 | 设计文档已过时，需要根据最新架构更新 |
