## 📋 归档说明

本目录存放基于v1.0架构设计的AI相关设计文档。这些文档已被新的v2.1架构设计文档取代。

**v2.1架构主要变化**：
- 引入Python AI Agent系统（LangChain、工具调用、RAG集成）
- Go/Python分层架构（Go作为API代理层，Python处理AI核心逻辑）
- 强制流式接口（降低用户感知延迟）
- 事件驱动架构完善（强制事件发布）
- Saga事务模式（保证分布式事务一致性）
