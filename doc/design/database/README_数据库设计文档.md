## 📊 数据模型

### 核心实体

- **用户 (User)** - 用户基本信息、认证信息
- **项目 (Project)** - 写作项目、作品信息
- **文档 (Document)** - 文档内容、章节信息
- **书籍 (Book)** - 书城书籍、详情信息
- **分类 (Category)** - 书籍分类、标签系统

### 关系设计

```tree
User (1) ─────── (N) Project
             │
             └─── (N) Book

Project (1) ───── (N) Document

Book (1) ────── (N) Chapter
         │
         └────── (N) Rating
```
